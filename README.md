# Phishing Awareness Simulation

## Project Overview
This project demonstrates a phishing awareness simulation conducted in a controlled environment to educate users about phishing attacks and cybersecurity risks.

## Objectives
- Understand phishing techniques
- Measure user awareness
- Analyze phishing risks
- Recommend preventive measures

## Tools Used
- HTML
- CSS
- Google Forms
- GitHub

## Project Components
- Fake Login Page (Educational Demo)
- Phishing Email Template
- Awareness Survey
- Results Analysis
- Preventive Measures

## Conclusion
User awareness and cybersecurity training play a crucial role in preventing phishing attacks.
